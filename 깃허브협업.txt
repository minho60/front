깃 & 깃허브 협업하기

팀장: 
    1. 깃허브 접속 후 레퍼지토리 만들기 (예: team)
    2. 세팅 - 콜라보레이터 - add people에 참가할 인원 깃허브 아이디입력
    3. git 폴더 통합터미널 열기 -> git clone (깃주소)https://github.com/minho60/team.git 
    4. 클론하고 cd (폴더이름)
    5. 브랜치 생성 - 통합 터미널-> git checkout -b (브랜치 이름)
    6. 파일 올리기 - 통합 터미널 git add . -> git commit -m "내용" -> git push origin kmh
         


팀원:
    1. 협업 요청 이메일확인
    2. view invitiation 클릭 / 허용
    3. 레파지토리 로컬 복제 code 주소 복사 
    4. vs code 에서 터미널 열기
    5. git clone (깃주소)https://github.com/minho60/team.git 입력하고 클론 파일 생성
    6. 복제한 폴더로 이동
    7. 브랜치 생성 통합 터미널-> git checkout -b (브랜치 이름)
    8. 임의 파일 생성 
    9. 개별 브랜치에 업로드 
        - git add .
        - git commit -m "커밋내용 "
        - git push origin 개인브랜치명 


    main 브랜치로 병합하기(Pull Request, PR)
    PR 생성하기
    base: main
    compare: 개인브랜치

    개인 브랜치에서 Open a pull request 클릭 base:가 main인지 반드시 확인
    팀장이 Pull Request 확인 후 main에 업로드 수정사항 메세지 커밋 가능
